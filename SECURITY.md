# Security Policy

## Supported versions

The latest reviewed `0.1.0-beta.*` receives security fixes after its protected
prerelease is published. Older prereleases and `0.0.x` are unsupported. Stable
release notes will name any additional supported line.

## Reporting

Report suspected vulnerabilities through GitHub private vulnerability
reporting:

https://github.com/global-torque/design-tokens/security/advisories/new

Do not open a public issue or include credentials, customer data, private URLs,
or unpublished vulnerability details in public artifacts. An owner should
acknowledge a complete report within five business days and provide a triage
decision or request for additional reproduction details within ten business
days.

This repository does not advertise an email security address until that mailbox
has been independently verified.
